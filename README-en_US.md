# omodule
`omodule`是一种帮助开发者更好的管理项目模块的文件目录管理规范。

## The omodule scope

#### __onamespace
- &lt;string&gt;

namespace of current omodule

#### __oname
- &lt;string&gt;

name of current omodule

#### __ofile
- &lt;string&gt;

name of current file

#### __ofilepath
- &lt;string&gt;

path of current file base on omodule root node
